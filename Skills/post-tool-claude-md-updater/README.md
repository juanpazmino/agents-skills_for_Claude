# post-tool-claude-md-updater

A drop-in folder that adds a `PostToolUse` hook to Claude Code. After every `Write` or `Edit` tool call, it uses the Claude API to generate a one-line summary of what changed and prepends it to your project's `CLAUDE.md` under a `## Recent Changes` section.

## Folder Structure

```
your_project/
тФЬтФАтФА .claude/
тФВ   тФФтФАтФА settings.example.json   # Hook registration (contains $PWD placeholder)
тФВ   тФФтФАтФА settings.local.json     # Generated by init script (actual absolute path)
тФЬтФАтФА hooks/
тФВ   тФФтФАтФА update_claude_md.js     # Hook logic тАФ summarizes changes and updates CLAUDE.md
тФЬтФАтФА scripts/
тФВ   тФФтФАтФА init-hook.js            # Setup script тАФ resolves $PWD to an absolute path
тФФтФАтФА package.json                # Dependencies + init script
```

## Usage

### 1. Copy the folder into your project root

```bash
cp -r post-tool-claude-md-updater/hooks        your_project/hooks
cp -r post-tool-claude-md-updater/.claude      your_project/.claude
cp -r post-tool-claude-md-updater/scripts      your_project/scripts
cp    post-tool-claude-md-updater/package.json your_project/package.json
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the setup script

From your project root:

```bash
npm run init
```

This replaces the `$PWD` placeholder in `.claude/settings.example.json` with the real absolute path of your project and writes it to `.claude/settings.local.json`.

You should see:

```
тЬЕ Successfully created .claude/settings.local.json
   Replaced $PWD with: /path/to/your/project
```

## How it works

After every `Write` or `Edit` tool call, `update_claude_md.js` receives the tool payload via stdin, calls the Claude API (Haiku model) to generate a concise summary of the change, then inserts it at the top of the `## Recent Changes` section in your project's `CLAUDE.md`.

The result looks like this in your `CLAUDE.md`:

```markdown
## Recent Changes

- **2026-03-06** `src/auth.js` тАФ Added JWT authentication middleware
- **2026-03-06** `src/db.js` тАФ Replaced raw SQL queries with Knex query builder
```

**Safeguards built in:**
- Skips edits to `CLAUDE.md` itself to prevent infinite loops
- Skips `.git/` internals
- Exits silently (code 0) on any error тАФ never blocks Claude's work
- Exits silently if `ANTHROPIC_API_KEY` is not set
- Never sends sensitive file content to the API тАФ blocks `.env`, `.envrc`, private keys, certs, SSH keys, and database files

## Requirements

- Node.js 18+
- `ANTHROPIC_API_KEY` available in your environment (Claude Code inherits it automatically)
